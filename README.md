# Todo List ver API

此專案可透過 API 進行使用者的身份認證，並處理待辦事項的新增、修改、刪除等等

![img](https://i.imgur.com/SoSuIqf.jpeg)

## Feature

- 創立帳號
- 登入 / 登出
- 新增待辦事項
- 修改待辦事項
- 刪除待辦事項

## Project Setup

```sh

```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
